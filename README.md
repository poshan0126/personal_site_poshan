### DevSwarm
**HiVE Engineering Platform**  
Helped build and ship DevSwarm, a parallel AI coding platform enabling developers to run multiple AI assistants (Claude, Gemini, Amazon Q) concurrently across isolated Git branches. Revolutionized development workflows with HiVE (High Velocity Engineering) approach, cutting feature delivery time from 4-6 hours to under 30 minutes.

*Tech: TypeScript, Electron, Git Worktrees*

### ThinkFolio
**RAG Application**  
AI-powered research paper interaction platform built with Next.js 15, enabling researchers to upload PDFs and engage in intelligent conversations with academic documents. Features streaming AI chat responses with page-specific citations using RAG architecture and pgvector for semantic understanding.

*Tech: Next.js, FastAPI, Redis, OpenAI, Supabase, LangGraph, Vercel*
